# Templates

This directory contains template projects demonstrating FHEVM SDK integration.

## Available Templates

### Next.js Template
Location: `../examples/nextjs-example/`

A complete Next.js application showcasing:
- FHEVM SDK integration
- React hooks for encryption/decryption
- API routes for server-side operations
- UI components for FHE operations
- Example use cases (banking, medical records)

**Quick Start:**
```bash
cd ../examples/nextjs-example
npm install
npm run dev
```

### Node.js Template
Location: `../examples/nodejs-example/`

A pure Node.js CLI application demonstrating:
- Server-side FHEVM operations
- Encryption and decryption
- Contract interactions
- No frontend dependencies

**Quick Start:**
```bash
cd ../examples/nodejs-example
npm install
npm start
```

## Using Templates

All templates are production-ready and can be used as starting points for your own projects:

1. Copy the template directory
2. Install dependencies with `npm install`
3. Configure environment variables (see `.env.example`)
4. Start building your confidential application

## Template Structure

Each template includes:
- Complete SDK integration
- Configuration examples
- README with instructions
- Environment setup guides
- Example code and use cases

For more information, visit the main repository README.
